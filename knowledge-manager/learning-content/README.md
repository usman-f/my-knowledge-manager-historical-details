# Learning Content

Companion answers to `code-base-learning.md`. That file is the *plan* (what to read,
which Rust concepts, comprehension questions, exercises); these files are the
*content* (the answers, in-context Rust explanation, worked exercise solutions).

- One file per step: `step-NN-*.md`, keyed to the same step in the plan.
- Read the plan's step first (do the **Read** + form your own answers), then check
  here. Don't read these cold — the value is in attempting the questions first.
- Each file: **Understand** (answers to that step's "Understand before proceeding"),
  **Rust** (the step's concepts, anchored to real lines), **Try it** (solution code
  where the step has an exercise).
- Existing companions still apply: `glossary.md` (vocabulary), `borrow-checker-guide.md`
  (after Step 4), `first-change-walkthrough.md` (after Step 8).
